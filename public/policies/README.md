# Reviewed Policy PDF Folder

This folder is for policy PDFs that have been reviewed and approved for public
website publication.

Only add a PDF here when all of the following are true:

- the document owner has approved publication,
- version, review date, and next review date are confirmed,
- private business details and unpublished staff, client, or child information
  have been removed,
- the file is a final web-ready PDF, not a source document or review draft,
- `src/data/policies.ts` links it with a `/policies/*.pdf` path.

Do not commit unreviewed policy drafts, raw source documents, private policy
packs, or working copies. Keep those outside Git until they are approved for
public use.
