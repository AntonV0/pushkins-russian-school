# Contributing

Status: current as of 2026-05-04.

This project is currently maintained as a private product codebase rather than a
general open-source project.

Public issues and pull requests are not expected. If you are reviewing the
repository as a recruiter, collaborator, or technical reviewer, please treat it
as a portfolio case study rather than a reusable software package.

Do not submit or request the publication of:

- secrets, credentials, or local environment files
- private business documents
- raw source screenshots or migration material
- unpublished client, child, parent, or staff information
- unapproved school images, policy documents, fees, or timetable details

Approved changes should stay small, readable, and easy to audit.
